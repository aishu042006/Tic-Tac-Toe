# My React Vite Project 🚀

## 📌 Description
This is a React project built using Vite. It is fast, modern, and optimized for development.

## 🔥 Features
- Fast development with Vite
- React components
- Hot Module Replacement (HMR)

## 🛠 Tech Stack

- Vite
- JavaScript
- CSS

## 🚀 Getting Started

### 1. Clone the repository
git clone https://github.com/aishu042006/Tic-Tac-Toe

### 2. Install dependencies
npm install

### 3. Run the project
npm run dev
